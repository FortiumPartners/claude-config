Version 3.5.0 - October 15, 2025

Breaking Changes

  • API Parameter Renamed: The prompt parameter has been renamed to content for consistency across all API endpoints. Migration: Update all API calls to use content instead of prompt by November 15, 2025.
  • Deprecated max_tokens Removed: The deprecated max_tokens parameter has been removed. Use max_tokens_to_sample instead.

New Features

  • Extended Context Window: Claude now supports up to 200,000 token context windows, enabling analysis of entire codebases and long documents.
  • Vision Capabilities: New image analysis capabilities allow Claude to understand charts, diagrams, and screenshots.
  • Function Calling: Native function calling support for tool integration and structured outputs.

Performance Improvements

  • Faster Response Times: Response latency reduced by 25% through optimization of the inference pipeline.
  • Improved Throughput: API can now handle 50% more concurrent requests.

Security Updates

  • Enhanced Input Validation: Improved validation to prevent prompt injection attacks.
  • Rate Limiting: New rate limiting to prevent abuse and ensure fair usage.

Bug Fixes

  • Fixed issue where streaming responses would occasionally timeout.
  • Corrected Unicode handling for non-English text.
  • Resolved memory leak in long-running conversations.
